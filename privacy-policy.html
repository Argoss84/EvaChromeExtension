Politique de confidentialité

Extension : Evassistant

Dernière mise à jour : 2 juillet 2026

1. Objet

Cette extension Chrome (Evassistant) aide les utilisateurs connectés à EVA à consulter les participants de leurs sessions et à enregistrer des favoris de réservation.

2. Données collectées

L'extension ne collecte ni ne vend aucune donnée personnelle à des tiers.

Les données suivantes peuvent être traitées localement dans votre navigateur :

  • favoris de réservation (centre, jeu, nombre de joueurs, libellé) ;
  • informations de session EVA nécessaires à l'affichage des participants ;
  • jeton d'accès temporaire obtenu via votre session EVA déjà connectée.

3. Stockage

Les favoris sont stockés localement dans le navigateur (stockage de la popup de l'extension) sur votre appareil. Aucune base de données distante dédiée à l'extension n'est utilisée.

4. Communications réseau

L'extension communique uniquement avec les services EVA (app.eva.gg, www.eva.gg, api.eva.gg) pour récupérer les informations nécessaires à son fonctionnement.

5. Permissions

  • activeTab : accéder temporairement à l'onglet EVA actif lorsque vous ouvrez Evassistant ;
  • scripting : relayer les requêtes API via votre session EVA déjà connectée.

6. Partage des données

Aucune donnée n'est partagée avec des annonceurs ou des services tiers.

7. Suppression des données

Vous pouvez supprimer vos favoris depuis l'extension, ou supprimer toutes les données de l'extension en la désinstallant depuis Chrome.

8. Contact

Pour toute question relative à cette politique, contactez le développeur de l'extension.
